1) what is if statement?
  an if statement allows a program to make decision. if the condition is true, python execute the code inside the if block.

2) what is else statement ?
  else runs when the condition in the if statement is false.

3) what is elif statement? 
  elif is used to check another condition if the previous condition was false.

4) explain these operators
  - == is equal to.
  - != is not equal.
  - > is greater than.
  - < is less than.
  - >= is greater than or equal to.
  - <= is less than or equal to. 

5) what is the difference between = and ==?
  = is an assignment operator.    
  while == tells that the variable and the value are the same.
