1) To check Git version  =  git --version
2) To configure your username (one-time setup) = git config --global user.name "your username"
3) To configure your email (one-time setup) = git config --global user.email "your email"
4) To check Git configuration  =  git config --global --list
5) To initialize a Git repository  =  git init
6) To check Git status  =  git status
7) To add all files  =  git add . 
8) To create a commit  = git commit -m "your commit message"
9) To view commit history  =  git log --oneline
10) To connect your project to GitHub (only once)  =  git remote add origin "your github repository URL"
11) To rename the branch to main (only once)  =  git branch -M main
12) To upload your code to GitHub:
    The first time  =  git push -u origin main
    After that  =  git push
